Export og-image.png — Owner Action Required

To complete the social sharing card setup, export the SVG below as a PNG file named og-image.png and place it in the same folder as index.html.

  1. Open this file in Google Chrome or Firefox (double-click it).
  2. Right-click on the image below and choose Save image as...
  3. Save it as og-image.png in the same folder as index.html.
  4. Alternatively, take a screenshot of the image at exactly 1200×630 pixels using your browser's DevTools (F12 → device toolbar → set 1200×630 → screenshot).
  5. After saving og-image.png, you may delete this file (og-image-export.html).
