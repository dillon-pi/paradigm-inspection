   **script.js** at project root — vanilla JS only, under 50 lines (per NAV-03, UI-SPEC):
   ```javascript
   (function () {
     'use strict';

     var toggle = document.querySelector('.nav-toggle');
     var header = document.querySelector('.site-header');

     if (!toggle || !header) return;

     toggle.addEventListener('click', function () {
       var isOpen = header.classList.toggle('nav-open');
       toggle.setAttribute('aria-expanded', String(isOpen));
       toggle.setAttribute('aria-label',
         isOpen ? 'Close navigation menu' : 'Open navigation menu'
       );
     });

     // Close mobile menu when any nav anchor link is clicked
     var navLinks = document.querySelectorAll('#main-nav a[href^="#"]');
     navLinks.forEach(function (link) {
       link.addEventListener('click', function () {
         header.classList.remove('nav-open');
         toggle.setAttribute('aria-expanded', 'false');
         toggle.setAttribute('aria-label', 'Open navigation menu');
       });
     });
   })();
   ```
   Key constraints:
   - IIFE wrapper, `'use strict'`, no global scope pollution
   - `classList.toggle` returns boolean for aria-expanded
   - Anchor link clicks in mobile nav close the menu per UI-SPEC interaction contract
   - aria-expanded toggles "true"/"false", aria-label toggles open/close text
   - No ES modules, no external dependencies
